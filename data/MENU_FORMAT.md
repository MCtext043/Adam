# Формат меню для быстрого импорта

Самый удобный вариант — **файл `data/vk_menu.json`** (уже используется сайтом) или таблица **CSV / Excel**.

## Вариант 1: JSON (рекомендуется)

Сохраните файл как `data/vk_menu.json`:

```json
[
  {
    "name": "Хачапури по-аджарски",
    "description": "Сыр сулугуни, масло, желток. 400 г",
    "price": "450",
    "category": "Выпечка",
    "image_url": "https://example.com/photo.jpg"
  }
]
```

| Поле | Обязательно | Пример |
|------|-------------|--------|
| `name` | да | Название блюда |
| `description` | да | Состав, вес, кратко |
| `price` | да | `450` (рубли, без ₽) |
| `category` | нет | `Горячее`, `Салаты`, `Напитки`… |
| `image_url` | да | Прямая ссылка на фото |
| `vk_id` | нет | Только если импорт с VK; для ручного меню не нужен |

Без фото и с ценой 0 позиция не попадёт в каталог.

После присылки файла: положите в `data/vk_menu.json` и задеплойте — меню подтянется при старте.

## Вариант 2: CSV (для Excel / Google Таблиц)

Кодировка **UTF-8**. Первая строка — заголовки:

```csv
name;description;price;category;image_url
Хачапури по-аджарски;Сыр сулугуни, 400 г;450;Выпечка;https://...
Шашлык из баранины;Баранина, лук, 250 г;920;Горячее;https://...
```

Разделитель **`;`** (удобно для русского Excel). Можно прислать `.xlsx` — конвертируем в CSV.

## Вариант 3: Google Таблица

Одна вкладка, столбцы как в CSV. Достаточно ссылки «доступ по ссылке — просмотр».

## Фото

- Лучше **прямые URL** (`https://…jpg`).
- Можно папку в **ZIP** (`название блюда.jpg`) — сопоставим по имени.
- Фото с VK уже в `data/vk_menu.json` после скрапинга.

## Что прислать в чат

1. Файл **JSON** или **CSV**, или ссылку на таблицу.  
2. Если меняете только цены — CSV с колонками `name` и `price`.  
3. Для полной замены меню — все поля из таблицы выше.

Команда на сервере (если правите вручную):

```bash
python scripts/import_vk_menu.py
```
